# PORTZEN SEO Stratejisi 2026 — Niş Anahtar Kelime Hedefleme

**Amaç:** Rakiplerin göz ardı ettiği long-tail, düşük rekabet, yüksek satın alma niyetli (transactional intent) keyword'lerle 6 ay içinde aylık **500 organik lead** hedefi.

**Kural:** Fiyat/paket/₺ HİÇ geçmez. AI araç adları HİÇ geçmez.

## 📊 Mevcut Durum Analizi

**Ne yaptık (envanter):**
- 95 blog yazısı (`content/blog/`)
- 125 sektörel landing (5 hizmet × 25 sektör = `content/landings/`)
- 15 karşılaştırma sayfası (`*-alternatifi`)
- 15 ana hizmet hedef sayfası
- 1 karşılaştırma index (`/karsilastirma`)
- Sitemap otomatik + Schema.org + hreflang tr-TR

**Nerede zayıfız (fırsat):**
1. **Coğrafi long-tail** — "cihangir restoran whatsapp botu", "kadıköy klinik randevu sistemi" gibi mahalle+sektör+problem kombinasyonları yok
2. **Sektörel mikro-problem** — "eczane iş takip whatsapp", "veteriner aşı hatırlatma sms" gibi 4-5 kelimelik intent zincirleri kapsanmamış
3. **Problem-çözüm formatlı blog** — "personel yetişemiyor whatsapp botu nasıl kurulur" tarzı direkt satın alma yolculuğu tetikleyen içerik az
4. **Karşılaştırma yerleşik değil** — "AI video prodüksiyon vs geleneksel çekim" tarzı direkt karşılaştırma daha az var
5. **Yerel işletme sinyalleri** — Google Business, LocalBusiness schema, mahalle/sokak referansı zayıf

## 🎯 SEO On-Page Checklist (yeni sayfa üretirken)

Her yeni niş sayfa için minimum:

- [ ] H1 — target keyword tam eşleşme (60 karakter max)
- [ ] Title tag — keyword + brand + benefit (60 kar)
- [ ] Meta description — 155 karakter, CTA'lı
- [ ] URL slug — kısa, keyword içerir, tire ayrımı
- [ ] Canonical link
- [ ] OG image (1200×630)
- [ ] Schema.org — Service, FAQPage, BreadcrumbList
- [ ] İç link: 3-5 ilgili sayfaya link (Cornerstone content'ten)
- [ ] Dış link: 1-2 otorite referans (Google, Meta docs vb.)
- [ ] Görsel: en az 3 (alt text, next/image, WebP)
- [ ] TOC + jump link (uzun içerikte)
- [ ] Related posts sonda
- [ ] CTA "Görüşme Al" en az 2 yerde
- [ ] Fiyat/paket YASAK
- [ ] AI araç adı YASAK
- [ ] KVKK/ETBIS uyum ifadeleri (güven sinyali)
